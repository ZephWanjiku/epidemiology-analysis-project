# Epidemiological Case Study Analysis

## 📌 Overview

This repository contains two integrated epidemiological case studies focused on infectious disease surveillance and outbreak investigation.

The analyses apply core public health and epidemiological methods including data cleaning, descriptive statistics, outbreak pattern analysis, and multivariable logistic regression modeling.

The projects aim to demonstrate the use of routine surveillance and laboratory data to generate actionable insights for disease monitoring and control.

---

## 📁 Projects Included

### 1. MPOX_OUTBREAK_ANALYSIS
This project focuses on Mpox (Monkeypox) outbreak surveillance data.

**Key objectives:**
- Describe temporal trends of Mpox cases
- Analyze geographic distribution of cases across districts
- Assess demographic patterns (age and sex distribution)
- Identify reporting delays in the surveillance system

**Main outputs:**
- Epidemic trend summaries
- District-level case distribution
- Descriptive epidemiological insights

---

### 2. MULTI_DISEASE_SURVEILLANCE_ANALYSIS
This project analyzes multiple notifiable diseases using integrated surveillance and laboratory datasets.

**Key objectives:**
- Integrate notification and laboratory datasets
- Clean and structure surveillance data for analysis
- Identify factors associated with confirmed disease cases
- Apply logistic regression to estimate odds ratios for key predictors

**Main outputs:**
- Data linkage between notification and laboratory systems
- Disease pattern analysis across districts
- Logistic regression models identifying risk factors for confirmed cases

---

## 🧪 Methods

The following epidemiological and statistical methods were applied:

- Data cleaning and preprocessing
- Handling missing data in surveillance systems
- Feature engineering (age groups, reporting delays, case classification)
- Descriptive epidemiology (time, place, person analysis)
- Data integration (case notification + laboratory confirmation)
- Multivariable logistic regression modeling
- Odds ratio estimation with confidence intervals

---

## 📊 Key Variables Used

- Age group
- Sex
- District of residence
- Disease type
- Pregnancy status (where available)
- Reporting delay (onset to notification)
- Laboratory test results

---

## ⚙️ Tools and Technologies

- R programming language
- Data manipulation and visualization (dplyr, ggplot2)
- Statistical modeling (glm, broom)
- Epidemiological surveillance datasets
- Git & GitHub for version control

---

## 📌 Key Insights

- Disease burden varies significantly across districts and demographic groups
- Reporting delays are a major limitation in timely surveillance response
- Laboratory confirmation improves accuracy of case classification
- Certain demographic and clinical factors are associated with increased likelihood of confirmed infection

---

## ⚠️ Limitations

- Missing data in key variables affected completeness of analysis
- Some predictors were excluded due to lack of variation after data cleaning
- Surveillance data quality depends on reporting completeness
